# Unity-RPG [Unity 2D]
Unity 엔진을 기반으로 제작된 2D RPG게임으로 포트폴리오 제출용으로 제작되었습니다.
# 1. 소개 < 게임 플레이 사진 > 
- 유니티 엔진을 활용해 제작한 2D 포트폴리오 입니다.
- 개발 기간 : 2023.07 ~ 2024.08 (약 1개월)
# 2. 개발 환경
- C#
- Window 10
